# 🗂️ Kanban Task Manager

A simple and interactive **Kanban Board** built using **HTML, CSS, and JavaScript**.
This project allows users to manage tasks efficiently with drag-and-drop functionality and persistent storage.

---

## 🚀 Features

* ✅ Add new tasks with title and description
* 🗑️ Delete tasks easily
* 🔄 Drag & drop tasks between columns (To Do, Progress, Done)
* 💾 Data persistence using Local Storage
* 🔢 Real-time task count update for each column
* 🧾 Input validation for better user experience
